## 📋 Core Concepts

### Boards
- **Regular Boards:** Shared project boards with 3 columns (Assigned → In Progress → Complete)
- **Parking Lot:** Your personal backlog for ideas and future tasks

### Cards
Cards represent tasks or work items. They flow through columns as work progresses.

---

## 🎯 Common Tasks

### Create a Board
1. Dashboard → **+ New Board**
2. Enter name and description
3. Click **Create Board**

### Add Cards
1. Open a board
2. Click **+ Add Card** in any column
3. Enter title and create
4. Click the card to add details (description, deadline, checklist, etc.)

### Move Cards
**Three ways:**
- **Drag & drop** cards between columns
- **Arrow buttons** (← →) on cards
- **Undo** with `Ctrl+Z` (or Cmd+Z) within 10 seconds

### Assign Work
1. Open a card
2. **Assignee** section → select team member
3. Click **Assign**

### Track Progress
1. Open a card
2. Add **checklist items** for sub-tasks
3. Check items off as you complete them
4. Add **comments** to discuss work
5. Update **Time Estimate** and **Actual Time**

### Set Deadlines
1. Open a card
2. Set **Completion Deadline**
3. Cards past deadline show red warning badge
4. Column headers show overdue count

### Move from Parking Lot
1. Open a card in your Parking Lot
2. **Move to Board** section → select destination board
3. Choose column (Assigned/In Progress/Complete)
4. Click **Move**

## 👥 Collaboration

### Share a Board
1. Open board → **Settings**
2. **Add Member by Username**
3. Type username → **Add Member**

### View Activity
- Board → **Activity** button
- See who created, moved, or assigned cards

### View Metrics
- Board → **📊 Metrics** button
- See completion rates, time tracking, assignee performance
- Dashboard → **📊 View Analytics** for overview

## 🔍 Card Details

### Priority
- **High:** Urgent work (red badge)
- **Medium:** Normal priority
- **Low:** Nice-to-have (blue badge)

### Tags
Add comma-separated tags: `bug, urgent, frontend`

### Checklists
- **Add items** with **+ Add Item**
- **Indent/Outdent** (→ ←) to create sub-items (5 levels max)
- **Check off** when complete
- **Click text** to edit inline

### Comments
- Add updates, questions, or notes
- Edit/delete your own comments
- See comment history with timestamps

### Time Tracking
- **Time Estimate:** How long you think it'll take
- **Actual Time:** How long it actually took
- View accuracy in Metrics

---

## 📊 Understanding Metrics

### Dashboard Analytics
- **Completion Rate:** % of cards marked complete
- **Cards This Week:** Activity in last 7 days
- **Estimate Accuracy:** How close estimates are to actual time

### Board Metrics
- **Column Distribution:** Where work is stuck
- **Assignee Performance:** Who's completing what
- **Activity Trend:** Work completed over time

## 💡 Tips & Tricks

### Efficient Workflows
1. **Start in Parking Lot:** Capture ideas without commitment
2. **Move to Board:** When ready to work, move to project board
3. **Assign & Prioritize:** Give cards to team members with priority
4. **Track Time:** Estimate before starting, log actual when done
5. **Use Deadlines:** Red badges keep urgent work visible
6. **Review Metrics:** Weekly check-in on team velocity

### Keyboard Shortcuts
- `Ctrl+Z` (or `Cmd+Z`): Undo last card move

### Best Practices
- **Clear titles:** "Fix login bug" not "Bug"
- **Use checklists:** Break complex work into steps
- **Comment progress:** Keep team informed
- **Archive when done:** Keep boards clean (View Archived)
- **Set realistic deadlines:** Red badges lose meaning if everything's overdue

### Managing Your Parking Lot
- Brain dump ideas here without clogging project boards
- Review weekly, move ready items to active boards
- Tag items by theme: `ideas, research, later`

## ❓ Troubleshooting

**Can't log in?**
- Check username/password spelling
- Contact board owner to verify account

**Can't see a board?**
- Ask board owner to add you in Settings

**Card won't move?**
- Try refreshing the page
- Use arrow buttons instead of drag-drop

**Deadline not showing?**
- Make sure you clicked **Save Changes** after setting deadline

## 🎨 Interface Tips

**Color coding:**
- **Red badge/border:** Overdue cards
- **Orange column:** In Progress
- **Green column:** Complete
- **Avatar circles:** Assignees

**Card badges:**
- ⏱️ = Time estimate
- 📅 = Deadline
- ⚠️ = Overdue
- ✓ = Complete
